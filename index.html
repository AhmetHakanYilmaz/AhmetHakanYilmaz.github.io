Projects

Paper Atlas

Senior Design Project for the bachelor's degree. In frontend React and Cytoscape.js, in backend node.js and a Neo4j database were used. The goal of this project is to enhance the way academicians and students conduct their research by providing a better way to find papers that best match their research topic by showing the hierarchy of papers related to the input paper of the user.

ClubVerse

Student club manager is a web-based application project for Object-Oriented Software Engineering (CS 319) course. Spring Boot, Jquery, Bootstrap, and PostgreSQL are technologies used in the project. The project aims to provide an effective implementation of a club manager for both sides of users, students and clubs, and makes it far easier to arrange and attend the club activities and follow a particular club.

Brain Tumor Detector

Brain tumor detector is Introduction to Machine Learning course project. We trained different models to detect 4 different classes. Logistic regression, CNN, MLP, Random forest, and transfer learning methods were used.

Aga Language

Aga language is designed for Programming Languages (CS 315) course. Designed a language which includes basic language needs. And built a parser for the language using the yacc tool.

Thread Scheduling Simulation

Implemented multithreaded scheduling simulation using mutex and condition variables to simulate different scheduling algorithms in Linux with C.

Online Coding Platform

Web-based group project for the database course. Spring Boot, React.js, and PostgreSQL are technologies used in the project. Developed an online coding website that includes different entities and relations. Worked on database design of the project.

Meeting Time Arranger

Developed a hobby mobile app using Flutter and Firebase Firestore to decide optimal meeting times with friends. Published at Google Play Store.

FAT 32 disk image

A program which displays different information from FAT32 disk images written in Linux with C.

Bus Ticket App

Term project for Cloud Computing (CS 443) course. For frontend Flutter was used, Node.js was used for backend, and a MongoDB database was used for the database part. Database and backend were deployed through Google Cloud Platform using App Engine. Simple ticket selling-buying application for bus companies and customers.

Text Categorizer

Text Categorizer is an Android project for Algorithms and Programming II (CS 102) course. It was developed with Java and Firebase Realtime Database was used. It allows users to find the important keywords that indicate the contexts of their notes and use those keywords to categorize the notes. It was created to ease users' workload when they need to get notes in a specific context.
